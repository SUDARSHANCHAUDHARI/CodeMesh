# Changelog

All notable user-facing changes are documented here.

## Unreleased

- Added post-launch backlog after the `0.4.0` public beta.

## 0.4.0 - Public Beta - 2026-07-04

- Added public install, configuration, command, privacy, security, and contribution documentation.
- Added portable config initialization for non-Sudarshan workspaces.
- Added `codemesh --version`.
- Added CLI smoke checks.
- Added GitHub Actions CI.
- Added public package metadata and MIT license.

## 0.3.0 - Private Local-First Foundation

- Added first-party plugin registry.
- Added optional read-only GitHub, GitLab, and Bitbucket repository providers.
- Added local Markdown, Obsidian, Notion export, NotebookLM export, and GitHub Wiki export knowledge detection.
- Added local dashboard generation.
- Added daily, weekly, release-note, changelog, PR summary, repository comparison, and usage summary reports.
- Added local memory, AI usage tracking, and knowledge graph commands.
- Added local plugin manifest validation and automation command plans.
- Added multi-agent instruction detection for Claude, Codex, Gemini CLI, OpenCode, Aider, Amp, Cursor, and Windsurf.

## 0.2.0 - Repository Portfolio Tools

- Added repository search, category, language, framework, and source filters.
- Added local-only, remote-only, duplicate, missing-local, and missing-remote views.
- Added repository provider comparison with JSON output and likely matching.
- Added clone plan commands for unresolved GitHub-only repositories.
- Added repo show, path, open, cd, dirty, stale, and summary commands.
- Added capsule preview and capsule history commands.
- Improved repository metadata with language, framework, package manager, branch, dirty status, and last commit date.

## 0.1.0 - MVP

- Added TypeScript CLI scaffold.
- Added local configuration.
- Added category-based local repository discovery.
- Added SQLite repository index.
- Added read-only Obsidian structure detection.
- Added Claude and Codex instruction detection.
- Added Markdown context capsule generation.
- Added `doctor`.
- Added basic docs.
